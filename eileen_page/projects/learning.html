sites/books/app that I recommend

Mimo (app):

This is the application that got me to start learning description languages and then learn programming languages. I bought the premium version during the 2020 lockdown. Thanks to this I learned CSS, HTML and a bit of JavaScript. Python and SQL were not yet available on the application. This application has a good graphical interface that is really easy to use. We can see on the application the site that we are coding live and it even allows us to publish it in order to share it. There is also a certificate system. I recommend this application only for HTML and CSS, I didn't really like learning Javascript on it and I think for learning Python and SQL there are other better tools.

https://www.w3schools.com/

This site helped me a lot for learning HTML, CSS, Javascript, Python,... It is very complete, has tutorials and exercises to learn. I really like the way it's organized.

Liftoff

Liftoff is a good flight simulator for learning to fly a drone. There are several different maps and games, you can customize your drone, the maps, and you can fly with other users.
